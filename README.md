# Super50 Web Development

Super50 Web Development is a project-based repository focused on building strong foundations in modern web development. It includes hands-on practice with core web technologies and helps in understanding how real-world web applications are structured.

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6+)  
- Visual Studio Code  


## Purpose

This repository is intended for:
- Learning web development from scratch  
- Practicing frontend concepts  
- Building real-world mini projects  
- Strengthening JavaScript fundamentals
